# Using PWAs

Pogressive Web Applications are used to provide application support to many devices with only one codebase. OpenLoops PWA is compatable with Firefox, Chrome and Safari (iOS only).

### Starting out on Desktop

Chrome allows PWAs to be installed into an app interface, on firefox there will still be a PWA it just wont look like a native app.
